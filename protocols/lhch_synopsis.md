# LHCH Prospective Validation — Synopsis
Population: Adults attending Hypertension Clinic; retinal images + routine eGFR/ACR from EHR.
Index test: RETFound+DeepSurv ESRD risk (2y/5y).
Comparator: KFRE (± biomarker change if longitudinal data present).
Outcomes: ESRD, ≥40% eGFR decline, CKD stage transition.
