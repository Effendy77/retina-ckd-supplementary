# Retina–CKD Supplementary Package

This repository contains **supplementary materials** for the EHJ Digital Health manuscript on predicting **ESRD/time-to-event** from retinal images using a RETFound+DeepSurv model and comparisons against KFRE.

It is designed to be linked from the main code repo: https://github.com/Effendy77/retina-esrd-survival

## What’s included
- `SUPPLEMENTARY.md` — human‑readable index of all supplemental items with short summaries.
- `manuscript/` — LaTeX + Word templates for main + supplement.
- `figures/` — auto‑generated, publication‑ready figures with scripts.
- `tables/` — CSV and TeX for key tables (baseline, performance, reclassification).
- `protocols/` — prospective LHCH validation synopsis and checklist.
- `checklists/` — TRIPOD‑AI, STROBE, and CONSORT‑AI (where applicable) placeholders.
- `references/` — `refs.bib` and PDF parking area (add your licensed PDFs locally).

> Tip: keep heavy PDFs in a **GitHub Release** or use Git LFS; do not bloat the repo.
